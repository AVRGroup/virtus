SOBRE O PROJETO

O programa Virtus tem como principal objetivo reunir iniciativas que tornem possível a aproximação de alunos de nível fundamental e médio à área de ciências exatas, promovendo a inclusão digital de grupos sociais vulneráveis.

O programa também visa disponibilizar projetos com foco na aplicação de recursos tecnológicos na área de saúde e telemedicina.

VRTOOLS

O VRTools tem como principal objetivo disponibilizar aplicações educacionais baseadas em Computação Gráfica (CG), Realidade Virtual (RV) e Realidade Aumentada (RA). O projeto disponibiliza também tutoriais de como utilizar ferramentas básicas de RV e RA para usuários novatos. Este projeto é a porta de entrada para alunos de diferentes níveis no entendimento das potencialidades de uso de recursos gráficos na educação.

OrBI

Um dos maiores problemas relacionados à popularização da Realidade Virtual é o custo dos equipamentos necessários para criar imersão, tornando sua adoção impraticável na maioria das instituições de ensino nos países em desenvolvimento. O projeto OrBI (Orbit-Based Interface) visa criar mecanismos que possibilitam o uso de smartphones e óculos de RV, ambos de baixo custo, na criação e uso de experiências imersivas em RV.

 Site do Projeto

FISIOVR

O principal objetivo deste projeto é prover uma arquitetura, baseada em visão computacional, que possibilite a individualização do tratamento de reabilitação à distância de um determinado indivíduo, com feedback em tempo real para o fisioterapeuta e/ou educador físico responsável pelo tratamento. Este projeto tem o potencial de beneficiar indivíduos com dificuldade de deslocamento que poderiam fazer suas atividades físicas a domicílio, com a supervisão de um profissional de saúde.

Recyclos

Este projeto visa reaproveitar equipamentos, em especial computadores, não mais em uso no Instituto de Ciências Exatas, mas com potencial de uso em aplicações com menor demanda computacional como navegação na internet e edição de documentos. O projeto trará dois benefícios: 1) Capacitar bolsistas e voluntários do projeto em técnicas básicas de manutenção de computadores; 2) Fornecer equipamentos em condição de uso para utilização na UFJF, beneficiando laboratórios e grupos de ensino, pesquisa e extensão associados (aplicação piloto inicial) e posteriormente beneficiando instituições públicas previamente cadastradas.

ELoS

O projeto ELoS (Ensino de LOgica em Sistemas gráficos) visa criar ferramentas de ensino de lógica de programação através do uso de sistemas gráficos. A ideia é criar um sistema gamificado, onde alunos de diversos níveis possam ter seu primeiro contato com a lógica de programação de forma lúdica e divertida. Espera-se com este projeto atrair estudantes que, de outra forma, não considerariam uma carreira na área de exatas. Esta ação tem um interessante potencial de inclusão de grupos sociais menos privilegiados, uma vez que colabora no ensino de técnicas que podem ser posteriormente utilizadas para melhor posicionar os estudantes que optarem por uma carreira na área de exatas.
